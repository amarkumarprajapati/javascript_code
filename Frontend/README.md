# Frontend — Interview Prep Index

> React, CSS, HTML, Next.js, three.js for frontend interviews.

| # | Folder/File | Topic |

## Key topics
- **React**: JSX, virtual DOM, reconciliation, hooks (useState, useEffect, useMemo, useCallback, useRef, useContext), performance (memo, lazy), state management (Context, Redux Toolkit), lifecycle
- **Next.js**: SSR/SSG, routing, API routes, getServerSideProps/getStaticProps
- **CSS**: Flexbox, Grid, responsive design, CSS-in-JS
- **HTML**: semantic tags, forms, accessibility (ARIA), SEO basics

## How to revise
Read React notes → build small components → practice CSS layouts → answer Q&A out loud.
