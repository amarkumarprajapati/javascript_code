12. What is Semantic HTML?
Answer:

use of HTML elements for visual appearance, semantic HTML elements represent the structure and meaning
of the content on a web page. HTML uses semantics to reinforce the semantics or purpose of the content.
Through the use of semantic HTML, search engines and web developers can ensure a better understanding of
the content and its relevance. This often leads to improved search engine optimization, overall user experience
as well as improved accessibility.
For Example:

</strong> and <em></em> tags in semantic HTML.
