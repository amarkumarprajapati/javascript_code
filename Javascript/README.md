# JavaScript — Interview Prep Index

> This is the most important folder for MERN interviews. Study in order.

## Study order

| # | Folder | Topic | Status |
| --- | --- | --- | --- |
| 1 | `01-core/` | Scope, closures, `this`, hoisting, prototypes | core |
| 2 | `02-async/` | Event loop, promises, async/await | core |
| 3 | `03-advanced/` | Currying, debounce, deep clone, generators | advanced |
| 4 | `04-machine-coding/` | Polyfills + machine-coding rounds | advanced |
| 5 | `05-theory/` | OOP, DOM, browser APIs, control flow, error handling | theory |

## Files in `01-core/`
- `01-scope-hoisting.md` — execution context, hoisting, TDZ, var/let/const
- `02-closures.md` — closures + real use cases
- `03-this-binding.md` — the 4 rules of `this`, call/apply/bind, arrow fns
- `04-prototypes.md` — prototype chain, inheritance
- `05-coercion-equality.md` — `==` vs `===`, type coercion, truthy/falsy

## Files in `02-async/`
- `01-event-loop.md` — call stack, task vs microtask queue
- `02-promises.md` — promises, chaining, combinators
- `03-async-await.md` — async/await, error handling

## Files in `05-theory/`
- `Advanced Topics/` — prototypes, functional programming, memory management, performance
- `Browser APIs & Storage/` — localStorage, sessionStorage, cookies, fetch
- `Control Flow & Loops/` — loops, break/continue, switch
- `DOM Manipulation/` — DOM tree, selection, manipulation, events
- `Error Handling/` — try/catch, error types, custom errors
- `Objects & Object-Oriented Programming/` — objects, functions, OOP concepts
- `oops.txt` — OOP detailed notes
- `Eventloop.txt` — event loop deep dive

## How to revise
1. Read the `.md`.
2. Run the `.js` example files (`node filename.js`).
3. Close the notes and explain the concept out loud.
4. Tick the box in `ROADMAP.md`.
